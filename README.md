# Data cleaning example - Titanic dataset

This project is an example of a data-cleaning workflow, using the classic Titanic dataset, designed as a practical exercise for beginner data science students.

### Repository Structure

| Data_Cleaning_Titanic
|─ DataCleaningTitanic.ipynb
|─ README.md
└─ Titanic-Dataset.csv


### Modules needed

- Pandas
- NumPy
- Matplotlib

### Dataset Description

The Titanic dataset contains contains demographic and survival information of passengers aboard the RMS Titanic. It contains missing values, inconsistent entries, outliers, and irrelevant variables.
